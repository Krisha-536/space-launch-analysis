# Space Launch Analysis

This is a small project where I explored SpaceX launch data using Python.

## What I did
- Cleaned the dataset
- Checked success vs failure of launches
- Compared rockets based on success rate
- Looked at how success has changed over time
- Plotted graphs using matplotlib and seaborn

## Tech used
- Python
- pandas
- matplotlib
- seaborn

## How to run
pip install -r requirements.txt  
python analysis.py

## Notes
This was mainly to get comfortable with working on real datasets and visualizing them.
